MCCOMPILED

Bedrock-Edition Programming Language

Design, develop, and compile code using the most fleshed out command compiler for Minecraft: Bedrock Edition.

MCCompiled offers dozens of fleshed out features; typed variables with automatic tellraw construction, advanced inline operations, functions with parameters and return values, on-the-fly animation controller generation, /give with support for enchants/names/lore, and the list goes on.

Nearly two years of engineering and development packed into a small, 500KB compiler.
