OVERVIEW

MCCompiled is a programming language that shares a lot in common with vanilla commands but includes simple but extremely powerful features to lift the development burden. Syntax is straightforward, easy to read, but it can be as deep and complex as you want to make it.

Simplicity-First

Nothing in MCCompiled is without reason; it's designed from the ground-up with commands in mind. There's no complex syntax, and it's inspired by the best of existing languages.

SUBFUNCTIONS

Declare functions inside your MCCompiled code, which will automatically be sent to their respective files. Functions may have parameters and return values as well, making them useful for more than just code re-use.

VALUES

Ditch scoreboard objectives for MCCompiled's robust implementation on top of them called values. Values can use one of the built-in types, each of which has implementations for type conversion, user display, and arithmetic.

Values can be made global, forcing them to never be attached to a specific entity. Values can be operated with inline, with grouping, order of operations, and proper type conversion.

CONTROL FLOW

Instead of writing dozens of execute commands and subfunctions, MCCompiled allows you to use if-statements, else-statements, loops, etc. to control logic instead. It will automatically handle the creation/organization of subfunctions for you, and you can focus on what actually matters (logic).

Many common patterns are included, including checking for a certain number of entities, looping a certain number of times, looping over entities, or checking for specific blocks. Or, if you have a specific case, the vanilla execute command is still available with full language support.

DISPLAY

While we're ditching things, rawtext. MCCompiled uses a simple format-string system for inlaying values into your messages. Inlay values, selectors, and even perform operations, which will make a temporary variable for you automatically.

Auto Localization

Define a default language, and MCCompiled will automatically send any/all text that can be localized into the .lang file automatically. You'll never have to touch a lang file again, and you can have peace of mind that MCCompiled generates context-aware key names for manual editing, if needed.

METAPROGRAMMING

MCCompiled has an insanely dynamic metaprogramming system that can go as deep as you want to go. Automate with JSON files, construct and call functions dynamically, write macros for compile-time code reuse, and give those macros compile-time checks and assertions for bug-free guarantees.

DIALOGUE

Use MCCompiled's super simple dialogue format to write/manage dialogue directly inside your project files. It integrates seamlessly with automatic localization support and inline MCCompiled code for event responses.

You can even automate dialogue creation with metaprogramming, like with the rest of the language.

ASYNC

Starting today, you've written your last timer. MCCompiled has support for asynchronous functions which are compiled to nothing more than a timer and a bunch of subfunctions. You can wait for amounts of time, for conditions to be met, or for other async functions to finish.
